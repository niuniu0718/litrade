# 成本支撑分析

## 核心逻辑

碳酸锂价格不会长期跌破边际成本线。找到成本支撑位，就能判断价格下方空间。

## 成本结构

### 碳酸锂成本构成

```
碳酸锂成本 = 锂原料成本 + 加工成本 + 其他（能源、辅料、折旧）
```

### 不同工艺的成本曲线（从低到高）

| 工艺 | 原料 | 成本特征 | 大致成本区间 |
|------|------|---------|-------------|
| 盐湖提锂 | 卤水 | 固定投资大，可变成本低 | 最低，4-6万/吨 |
| 锂辉石提锂 | SC6精矿 | 取决于矿价，弹性大 | 中等，跟随矿价波动 |
| 锂云母提锂 | 锂云母矿 | 副产品可抵扣，但品位低 | 较高，成本波动大 |
| 再生回收 | 废旧电池 | 规模有限，成本偏高 | 较高 |

### 锂辉石提锂成本测算公式

```
碳酸锂成本（元/吨）= SC6精矿CIF价（美元/吨）× 汇率 × 8（吨矿/吨锂）× (1+损耗率) + 加工费
```

近似简化：**碳酸锂成本 ≈ SC6 CIF价 × 6.5-7.5 + 1.5-2万加工费**

### 关键变量
- SC6 CIF价格（美元/吨）：直接决定成本中枢
- 汇率（USD/CNY）：影响进口成本
- 加工费（TC）：与矿端议价有关

## 成本支撑分析方法

### Step 1：确定边际成本线
- 找到当前价格下，哪种工艺刚好不亏钱（边际生产者）
- 当前（2026年）边际成本线大约在**锂辉石提锂**这一档

### Step 2：计算边际成本
- 用最新SC6价格计算锂辉石提锂成本
- 这是价格的关键支撑位

### Step 3：判断支撑强度

| 信号 | 支撑强 | 支撑弱 |
|------|--------|--------|
| 价格 vs 边际成本 | 接近或跌破 | 远高于 |
| 高成本产能 | 已关停 | 还在运行 |
| 矿端利润 | 矿企也在亏损 | 矿企仍有利润 |
| 加工费 | 下调中 | 稳定或上调 |

### 输出

- 边际成本线：XX万元/吨
- 支撑强度：**强 / 中 / 弱**
- 方向判断：**偏多（接近成本有支撑）/ 中性 / 偏空（远高于成本）**
