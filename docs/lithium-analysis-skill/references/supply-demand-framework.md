# 供需平衡分析框架

## 方法论来源

融合五矿钢联、SMM、中信等机构的锂产业链供需平衡表分析逻辑。

## 分析维度

### 1. 供给端

#### 国内产量
- **周度产量**：SMM国内碳酸锂周度产量（吨），关注环比变化
- **月度产量**：同比增速判断供给扩张/收缩周期
- **开工率**：冶炼厂开工率，低于70%说明行业在主动减产
- **分工艺**：锂辉石提锂、锂云母提锂、盐湖提锂分别跟踪

#### 进口供给
- **智利出口**：SQM+Albemarle出口量，环比变化领先国内到港1-2个月
- **阿根廷出口**：盐湖项目增量
- **津巴布韦**：锂精矿出口配额变化
- **澳洲锂精矿**：SC6报价和发运量

#### 供给判断标准
| 信号 | 偏多（供给收紧） | 中性 | 偏空（供给宽松） |
|------|-----------------|------|-----------------|
| 周产量环比 | ↓ >2% | ±2%以内 | ↑ >2% |
| 同比增速 | <15% | 15-30% | >30% |
| 开工率 | <65% | 65-80% | >80% |
| 进口量环比 | ↓ >10% | ±10% | ↑ >10% |

### 2. 需求端

详见 `demand-outlook.md`

### 3. 供需平衡表构建

```
总供给 = 国内产量 + 进口量
总需求 = 正极材料产量 × 单耗 + 其他工业需求 + 出口
供需缺口 = 总供给 - 总需求（正值=过剩，负值=短缺）
```

#### 关键比率
- **库存消费比** = 社会库存 / 月度消费量
  - <0.8：偏紧
  - 0.8-1.2：平衡
  - >1.2：宽松

### 4. 输出

- 供需格局：**过剩 / 平衡 / 短缺**
- 置信度：高/中/低
- 关键边际变化：xxx环比变化xx，是主要驱动因素
