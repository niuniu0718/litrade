# 需求景气度分析框架

## 核心逻辑

锂需求的核心驱动是新能源汽车和储能，需跟踪终端→电池→正极材料的传导链条。

## 需求传导链

```
新能源车产销（月） → 动力电池装机（月） → 正极材料排产（周/月） → 碳酸锂需求
              ↗
储能装机（季） → 储能电池产量 → 磷酸铁锂需求 → 碳酸锂需求
```

## 关键指标与判断标准

### 1. 新能源汽车

| 指标 | 偏多 | 中性 | 偏空 |
|------|------|------|------|
| 月度销量同比 | >10% | 0-10% | <0% |
| 环比变化 | ↑ | 持平 | ↓ |
| 渗透率 | 持续提升 | 稳定 | 回落 |

### 2. 动力电池

| 指标 | 偏多 | 中性 | 偏空 |
|------|------|------|------|
| 装机量同比 | >15% | 0-15% | <0% |
| 磷酸铁锂占比 | 提升 | 稳定 | 下降 |

### 3. 正极材料排产

| 指标 | 偏多 | 中性 | 偏空 |
|------|------|------|------|
| 磷酸铁锂排产环比 | ↑ >5% | ±5% | ↓ >5% |
| 三元排产环比 | ↑ >5% | ±5% | ↓ >5% |

### 4. 储能

| 指标 | 说明 |
|------|------|
| 储能装机量 | 大型储能项目进度 |
| 储能电池出货 | 通常用磷酸铁锂，拉动碳酸锂需求 |

## 季节性规律

| 月份 | 需求特征 | 原因 |
|------|---------|------|
| 1-2月 | 偏弱 | 春节+淡季 |
| 3-4月 | 回暖 | 补库+排产提升 |
| 5-6月 | 稳定/偏强 | 年中冲量 |
| 7-8月 | 小淡季 | 高温假期 |
| 9-10月 | 强 | 金九银十 |
| 11-12月 | 强 | 年底冲量+备货 |

## 需求弹性判断

- **刚性需求**：长协、大客户固定订单 → 不随价格波动
- **弹性需求**：零单、投机性采购 → 价格涨了少买，跌了多买
- **关注点**：排产数据变化往往领先现货需求1-2周

### 输出

- 需求景气度：**强 / 稳定 / 弱**
- 边际变化方向：**改善 / 持平 / 恶化**
- 方向判断：**偏多 / 中性 / 偏空**
