# 价格预测方法论

## 核心原则

**不做单点预测，做多情景概率加权。** 参考西蒙斯的概率思维：不需要每次都对，只要概率对你有利，长期就能赢。

## 预测方法

### 1. 综合信号评分

将5个模块的判断汇总：

| 模块 | 偏多(+1) | 中性(0) | 偏空(-1) | 实际判断 |
|------|---------|---------|---------|---------|
| 供需平衡 | | | | |
| 库存周期 | | | | |
| 成本支撑 | | | | |
| 事件驱动 | | | | |
| 需求景气 | | | | |
| **合计** | | | | |

**评分规则**：
- +3~+5：强烈看多
- +1~+2：偏多
- 0：中性
- -1~-2：偏空
- -3~-5：强烈看空

### 2. 多情景构建

#### 基准情景
- 概率：50-70%
- 假设：当前趋势延续，无重大意外
- 价格区间：根据技术面+基本面确定

#### 上行情景
- 概率：15-30%
- 触发条件：列出具体事件（如"江西停产+津巴布韦延迟"）
- 价格目标：阻力位参考前期高点+情绪溢价

#### 下行情景
- 概率：15-30%
- 触发条件：列出具体事件（如"矿端扰动全部证伪+库存继续累"）
- 价格目标：支撑位参考成本线

### 3. 均价预测

```
预测均价 = 基准情景价格 × 基准概率 + 上行价格 × 上行概率 + 下行价格 × 下行概率
```

### 4. 时间维度

| 周期 | 方法 | 重点 |
|------|------|------|
| 1-2周 | 事件驱动+技术面 | 短期扰动和情绪 |
| 1个月 | 供需+库存 | 月度排产和到港 |
| 1个季度 | 供需平衡表+成本 | 趋势性判断 |
| 半年以上 | 产能周期+需求增长 | 结构性判断 |

### 5. 置信度标注

每次预测必须标注置信度：
- **高**：多个信号方向一致，数据充分
- **中**：信号有分歧，方向不够明确
- **低**：数据缺失或矛盾，不确定性大

### 6. 回顾与修正

- 每周回顾上周预测 vs 实际走势
- 如果偏差大，分析原因（哪个模块判断错了）
- 持续修正模型权重
