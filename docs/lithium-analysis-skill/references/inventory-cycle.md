# 库存周期与逼仓分析

## 方法论来源

大宗商品经典库存周期理论（基钦周期），结合广期所碳酸锂期货的仓单机制。

## 核心逻辑

库存是供需的"蓄水池"，库存趋势比绝对水平更重要。

### 1. 库存周期四阶段

| 阶段 | 价格 | 库存 | 供需 | 市场情绪 |
|------|------|------|------|---------|
| **被动去库存** | ↑上涨 | ↓下降 | 需求好转 | 乐观 |
| **主动补库存** | ↑上涨/高位 | ↑上升 | 供给跟进 | 谨慎乐观 |
| **被动补库存** | ↓下跌 | ↑上升 | 需求走弱 | 悲观 |
| **主动去库存** | ↓下跌/低位 | ↓下降 | 供给收缩 | 恐慌/筑底 |

**当前判断方法**：对比库存环比和价格环比的趋势

### 2. 关键库存指标

| 指标 | 数据来源 | 判断阈值 |
|------|---------|---------|
| 社会库存（吨） | SMM | 关注绝对值和环比方向 |
| 库存天数 | SMM | <20天偏紧，20-30天平衡，>30天宽松 |
| 冶炼厂库存 | SMM/调研 | 高=销售压力大 |
| 下游库存 | 调研 | 低=补库需求在 |
| 广期所仓单（吨） | 广期所 | 关注注册/注销趋势 |

### 3. 逼仓风险分析

碳酸锂期货逼仓条件：
- 可交割品不足（仓单 < 持仓量的合理比例）
- 某月合约持仓集中
- 现货升水大幅走扩

**逼仓预警信号**：
1. 仓单持续注销，总量下降
2. 某合约持仓量异常放大
3. 现货升水 > 2000元/吨
4. 库存天数 < 15天

**反逼仓信号**：
1. 仓单大幅注册（如单周+10%以上）
2. 现货贴水
3. 库存天数 > 30天且持续增加

### 4. 输出

- 库存周期阶段：**被动去库/主动补库/被动补库/主动去库**
- 逼仓风险：**高/中/低/无**
- 方向判断：**偏多 / 中性 / 偏空**
