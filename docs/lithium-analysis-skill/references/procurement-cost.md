# 综合采购成本测算

## 测算公式

```
综合采购成本（元/吨）= 
  [现货采购总金额 - 期货套保盈利] / 实际采购总量

其中：
- 现货采购总金额 = Σ(每笔采购量 × 采购单价)
- 期货套保盈利 = (期货卖出价 - 期货买入价) × 期货吨数
  注：买入套保时，盈利 = (期末价 - 均价) × 持仓量（未平仓为浮盈）
- 实际采购总量 = 现货采购量（期货是金融工具，不交割则为虚拟对冲）
```

## SMM折扣率

```
折扣率 = (1 - 综合成本 / SMM电池级碳酸锂均价) × 100%

目标：折扣率 ≥ 10%（即综合成本 ≤ SMM均价 × 90%）
```

## 测算示例

### 输入数据
| 项目 | 数值 |
|------|------|
| 月度采购目标 | 15,000吨 |
| 长协量 | 4,500吨 |
| 长协价 | 170,000元/吨 |
| 需套保量 | 10,500吨 |
| 期货多头 | 894手（4,470吨），均价161,000元 |
| 当前期货价 | 177,900元 |
| 零单现货预计采购 | 6,030吨 |

### 计算

```
1. 长协成本：4,500 × 170,000 = 7.65亿
2. 期货浮盈：4,470 × (177,900 - 161,000) = 7,554万元
3. 零单现货预估（假设均价175,000）：6,030 × 175,000 = 10.55亿
4. 综合成本 = (7.65亿 + 10.55亿 - 0.755亿) / 15,000 = 116,100元/吨
```

⚠️ 注意：如果期货最终平仓价与采购期间现货价走势一致，套保才能真正锁定成本。

## 风险提示

- 期货浮盈≠实际盈利，未平仓前都是账面利润
- 基差风险：期货与现货价格可能不同步
- 保证金风险：价格大幅波动可能需要追加保证金
- 交割风险：如选择实物交割，需确保有仓储物流能力
