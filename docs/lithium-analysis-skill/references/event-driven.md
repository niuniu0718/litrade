# 矿端扰动事件评估方法论

## 核心原则

**区分实质性影响与情绪炒作。** 大部分矿端扰动只影响短期情绪，不改变供需基本面。

## 事件分级标准

### Level 3：实质性影响（改变供需平衡）
- 大型矿山长期停产（>3个月）
- 主要出口国全面禁运
- 战争导致运输通道长期中断
- **判断标志**：能量化到具体减产吨数，且无法短期替代

### Level 2：短期情绪影响（1-4周，不改趋势）
- 环保督查临时停产
- 换证/审批延迟
- 短期运输受阻
- **判断标志**：有减产预期但时间不确定，或影响量<月产5%

### Level 1：无实质影响（噪音）
- 市场传闻未证实
- 单一矿山小规模减产
- 政策表态但无具体措施
- **判断标志**：无法量化影响，或已被市场充分定价

## 主要跟踪的矿端风险源

| 风险源 | 类型 | 关注指标 | 信息渠道 |
|--------|------|---------|---------|
| **澳大利亚** | 柴油供应/天气 | 燃油库存、发运量 | FuelSecurity、矿企公告 |
| **津巴布韦** | 出口政策 | 配额公布、出口许可 | 政府公告、贸易商 |
| **江西锂云母** | 环保/采矿证 | 换证进度、停产范围 | 地方政府、企业调研 |
| **智利/阿根廷** | 政策/天气 | 出口数据、政策变化 | 海关数据、新闻 |
| **非洲其他** | 政治风险 | 政局稳定性 | 新闻 |
| **中东地缘** | 油价/运费 | 原油价格、海峡通航 | 国际新闻 |

## 事件分析流程

```
事件发生
  ↓
1. 确认真实性（是传闻还是已确认？）
  ↓
2. 量化影响（影响多少吨？持续多久？）
  ↓
3. 评估替代性（其他供应源能否补上？）
  ↓
4. 判断级别（Level 1/2/3）
  ↓
5. 预判市场反应（已定价 or 还有空间？）
  ↓
6. 给出操作建议（该不该追/该不该跑）
```

## 经验法则（来自实战采购）

1. **"买预期卖事实"**：矿端扰动往往提前炒作，落地后反而回落
2. **大企业有专供**：柴油危机等事件，大矿山通常有独立供应渠道，受影响小
3. **看库存验证**：如果库存没变化，说明影响还没传导到现货
4. **关注恢复进度**：津巴布韦这类事件，关键是看恢复速度而不是暂停本身

### 输出

- 事件名称与级别：**Level 1/2/3**
- 影响量化：预计减产XX吨/月，持续X个月
- 市场定价程度：**已充分定价 / 部分定价 / 未定价**
- 方向判断：**偏多 / 中性（已消化）/ 偏空（证伪）**
