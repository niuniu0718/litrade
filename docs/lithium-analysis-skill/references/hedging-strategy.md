# 套保策略框架

## 核心原则

套保不是投机，目的是**锁定采购成本**，使综合成本低于SMM均价。

## 套保决策流程

```
明确敞口（需保值量）
    ↓
评估当前持仓（已有期货头寸）
    ↓
计算剩余敞口
    ↓
判断价格方向（来自价格预测模块）
    ↓
制定操作方案
```

## 持仓评估

### 需要的信息
| 项目 | 说明 |
|------|------|
| 月度采购目标 | 碳酸锂+氢氧化锂合计吨数 |
| 长协锁定量 | 已锁定价格和数量 |
| 需套保量 | 采购目标 - 长协量 |
| 已有期货持仓 | 合约、方向、手数、均价 |
| 剩余敞口 | 需套保量 - 已有期货覆盖量 |

### 持仓健康度检查

| 指标 | 健康 | 警告 | 危险 |
|------|------|------|------|
| 浮盈/浮亏 | 盈利 | 小亏 | 大亏接近止损 |
| 保证金占比 | <50% | 50-80% | >80% |
| 套保覆盖率 | 30-70% | <30%或>70% | 极端偏离 |

## 操作策略矩阵

| 价格判断 | 剩余敞口 | 操作 | 具体动作 |
|---------|---------|------|---------|
| **看涨** | 大 | 积极建仓 | 分批买入，每次200-300手 |
| **看涨** | 小 | 持有不动 | 不加仓，等价格验证 |
| **震荡** | 大 | 逢低买入 | 设定目标价位，到达后分批建仓 |
| **震荡** | 小 | 观望 | 现货按需采购 |
| **看跌** | 大 | 暂缓建仓 | 等价格跌到支撑位再买 |
| **看跌** | 小 | 考虑减仓 | 减仓100-200手锁利 |

## 风控规则

### 止损
- **成本线止损**：跌破持仓均价则止损
- **保证金止损**：单日浮亏超过保证金X%则减仓

### 止盈
- 不急止盈，让利润奔跑
- 但如果短期冲高过多（如单周涨>10%），可考虑部分止盈

### 仓位管理
- 单次建仓不超过总敞口的20%
- 总期货仓位不超过月度采购量的80%
- 留至少20%用现货采购，保持灵活性

## 综合成本测算

```
综合成本 = (现货采购量 × 现货均价 + 期货量 × 期货均价 - 期货盈亏) / 总采购量

SMM折扣 = (1 - 综合成本 / SMM均价) × 100%
```

**目标**：综合成本 ≤ SMM均价 × 90%（10个点折扣）

### 输出

- 当前持仓状态：浮盈/浮亏金额
- 操作建议：建仓/加仓/减仓/观望 + 具体手数和价位
- 预估综合成本：XX万元/吨
- SMM折扣预估：X%
